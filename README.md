# Lachik AI

Lachik AI is a modular Python framework for creating autonomous game agents and reusable game-AI systems.

> Current status: early development

## Goals

Lachik AI aims to provide:

- Engine-independent autonomous agents
- Reusable sensors, memory, goals, and actions
- Finite-state-machine decision systems
- Headless simulations
- Optional Pygame integration
- Extensible public interfaces
